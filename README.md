## Instruction

1. Clone this repo 
    ```bash
    git clone https://github.com/ravesjs/countries-app
    ```
     
    ## Installation
    
    Install dependencies
    ```bash
    npm install
    ```

    ### Development server
    ```bash
    npm start
    ```
    
    ### Production
    
    ```bash
    npm run build
    ```
